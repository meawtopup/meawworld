# 🔒 Host Script

สคริปต์สำหรับเจ้ามือจะถูกเก็บไว้ในเครื่องเท่านั้น
ไม่ได้อัพโหลดขึ้น GitHub เพื่อความปลอดภัย

## วิธี Setup

1. เปิด Tampermonkey Dashboard
2. สร้าง Script ใหม่
3. Copy script จาก local/meawworld-host.user.js
4. บันทึก

## ความปลอดภัย

- Host Script มี Secret Key เฉพาะเครื่อง
- ไม่สามารถใช้จาก GitHub ได้
- ป้องกันการปลอมตัวเป็นเจ้ามือ
